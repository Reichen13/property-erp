## 系统概述

物业ERP系统是一套完整的物业费用管理解决方案，支持：
- 房产档案管理
- 费用账单生成
- 收银收款
- 财务核对
- 审计追踪

### 用户角色

| 角色 | 权限范围 |
|------|----------|
| 管理员 | 所有功能，包括系统配置、用户管理 |
| 财务 | 财务管理、收费核对、报表查询、关账 |
| 收银员 | 收银台、欠费查询 |

## 基础概念

### 1. 复式记账

系统采用复式记账法，每笔交易同时记录借方和贷方：

```
充值：借方=现金(1)，贷方=预收账款(3)
缴费：借方=预收账款(3)，贷方=物业费收入(2)
```

**借贷平衡原则**：任何时刻，借方总额 = 贷方总额

### 2. 账期（Period）

格式：`YYYY-MM`（如 2026-01）

- 账单按账期生成
- 关账后该账期数据不可修改
- 跨账期操作需先解锁

### 3. 钱包余额

业主预存款项，可用于抵扣账单：
- 充值增加余额
- 缴费扣减余额
- 余额不足时需选择其他支付方式

### 4. 账单状态

| 状态 | 说明 |
|------|------|
| 待缴 | 新生成，未收款 |
| 部分已缴 | 已收部分款项 |
| 已缴 | 全额收款完成 |
| 作废 | 已取消，不计入统计 |

### 5. 核销（Write-off）

将收到的款项与账单进行匹配的过程：
1. 选择待缴账单
2. 确认支付方式
3. 系统自动更新账单状态和余额

### 6. 对账

确保数据一致性的检查过程：
- **收费核对**：按账期汇总应收/实收
- **三方核对**：业务数据 vs 会计科目 vs 实际资金
- **财务检查**：检测异常数据（负余额、超额缴费等）

## 系统初始化

### 步骤1：登录系统

1. 访问系统地址
2. 输入账号密码（默认：admin / admin123）
3. 首次登录后建议修改密码

### 步骤2：创建费用类型

路径：`系统管理 → 费用类型管理`

常见费用类型：
- 物业费（税率6%）
- 停车费（税率6%）
- 水费（税率3%）
- 电费（税率3%）

### 步骤3：导入房产档案

路径：`资源档案 → 批量导入`

Excel模板字段：
| 字段 | 说明 | 示例 |
|------|------|------|
| room_number | 房号 | 1-101 |
| owner_name | 业主姓名 | 张三 |
| area | 面积(㎡) | 89.5 |
| phone | 联系电话 | 13800138000 |

### 步骤4：设置收费标准

路径：`资源档案 → 收费标准`

为每个房产设置各费用类型的月度金额。

## 日常操作流程

### 场景1：业主充值

1. 进入 `收银台`
2. 搜索/选择房号
3. 展开"钱包充值"
4. 输入充值金额
5. 选择收款方式（微信/支付宝/现金/银行转账）
6. 点击"确认充值"

**系统自动执行**：
- 增加房产余额
- 生成收款记录
- 记录复式分录（借：现金，贷：预收账款）
- 写入审计日志

### 场景2：账单缴费

1. 进入 `收银台`
2. 选择房号
3. 在"待缴费账单"中勾选要缴费的账单
4. 选择支付方式：
   - 余额抵扣：从钱包扣款
   - 微信/支付宝：现场收款
5. 点击"确认支付"

**注意事项**：
- 余额不足时无法使用余额抵扣
- 可部分缴费（勾选部分账单）

### 场景3：批量生成账单

1. 进入 `财务管理 → 批量生成账单`
2. 选择费用类型或勾选"生成所有费用项目"
3. 输入账期（如 2026-01）
4. 点击"全量生成"

**生成逻辑**：
- 按档案预设金额：使用房产的收费标准
- 按单价x面积：使用输入的单价乘以房产面积

## 月度结账流程

### 第1步：生成当月账单

确保所有费用类型的账单已生成。

### 第2步：收费核对

路径：`收费核对 → 收费核对工作台`

1. 选择账期
2. 查看汇总数据：
   - 应收总额
   - 实收总额
   - 欠费总额
   - 收缴率
3. 检查明细，确认无异常

### 第3步：三方核对

路径：`收费核对 → 三方核对机制`

检查项：
- 房产余额总和 = 预收账款科目余额
- 账单应收总额正确
- 收款记录按支付方式统计

### 第4步：财务检查

路径：`收费核对 → 财务勾稽关系检查`

确保无以下异常：
- 负余额房产
- 超额缴费账单

### 第5步：关账

路径：`财务管理 → 月度关账`

1. 输入账期
2. 点击"关账"
3. 关账后该账期数据锁定

**解锁**：如需修改已关账数据，需先解锁账期。

## 常见问题

### Q1：账单金额错误怎么办？

1. 如账期未关账：直接修改账单
2. 如账期已关账：
   - 先解锁账期
   - 修改账单
   - 重新关账

### Q2：业主要求退款怎么处理？

1. 进入 `财务管理 → 减免审批`
2. 创建减免申请
3. 管理员/财务审批通过
4. 系统自动调整账单金额

### Q3：如何查看操作记录？

路径：`审计查询`

可按时间、用户、操作类型筛选查看所有操作日志。

### Q4：数据如何备份？

- **自动备份**：每日首次登录时自动备份
- **手动备份**：`数据备份 → 立即备份`

### Q5：忘记密码怎么办？

联系管理员重置密码，或使用数据库工具直接修改。
